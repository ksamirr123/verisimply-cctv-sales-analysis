# 📹 VeriSimply — UK B2B CCTV Sales Analysis

## Overview
Freelance data analyst project for VeriSimply, a UK-based B2B CCTV 
services company. Mapped the inquiry-to-sale conversion funnel 
and built a live Power BI dashboard for management.

**Tools:** SQL · Excel · Power BI  
**Type:** Real Freelance Client Work  
**Year:** 2025

---

## 🔍 Key Findings

| Finding | Detail |
|--------|--------|
| Top packages | Identified top 3 CCTV packages by conversion rate |
| Peak periods | Found high-demand inquiry periods by product type |
| Funnel analysis | Mapped full inquiry → sale drop-off points |
| Dashboard | Live management dashboard — single view of business health |

---

## 💡 Business Impact
- Management got live Power BI dashboard — no more manual Excel reports
- Inventory stocking aligned to high-demand periods
- Marketing spend redirected to peak inquiry windows
- Monthly trend reports replaced ad-hoc analysis
